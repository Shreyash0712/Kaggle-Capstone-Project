# Python Expert Guidance
- **Environment**: Always use a virtual environment (`venv`, `poetry`, or `conda`).
- **Type Hinting**: Use modern Python type hints (`typing` module) to improve code readability and catch errors early with `mypy`.
- **Testing**: Use `pytest` for unit testing. Write simple, isolated tests.
- **Code Formatting**: Use `black` and `isort` for formatting, and `flake8` or `ruff` for linting.
